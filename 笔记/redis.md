# 认识 NoSQL

| 对比项 | SQL | NoSQL |
|---|---|---|
| 数据结构 | 结构化 Structured | 非结构化 |
| 数据关联 | 关联的 Relational | 无关联的 |
| 查询方式 | SQL 查询 | 非 SQL |
| 事务特性 | ACID | BASE |
| 存储方式 | 磁盘 | 内存 |
| 扩展性 | 垂直扩展 | 水平扩展 |
| 使用场景 | 数据结构固定；对数据安全性、一致性要求较高 | 数据结构不固定；对一致性、安全性要求不高；对性能要求高 |

## NoSQL 类型

1. 键值类型：Redis
2. 文档类型：MongoDB
3. 列类型：HBase
4. Graph 图类型：Neo4j
